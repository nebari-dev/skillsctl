skillsctl has been deployed!

1. Check the deployment status:
   kubectl get pods -l app.kubernetes.io/name={{ include "skillsctl.name" . }},app.kubernetes.io/instance={{ .Release.Name }} -n {{ .Release.Namespace }}

2. Access the service:
{{- if .Values.ingress.enabled }}
   https://{{ (index .Values.ingress.hosts 0).host }}
{{- else if .Values.nebariapp.enabled }}
   https://{{ .Values.nebariapp.hostname }}
{{- else }}
   kubectl port-forward svc/{{ include "skillsctl.fullname" . }} 8080:{{ .Values.service.port }} -n {{ .Release.Namespace }}
   Then visit: http://localhost:8080/healthz
{{- end }}

3. Configure the CLI:
   skillsctl config set api_url <your-service-url>
{{- if not .Values.oidc.issuerURL }}

NOTE: OIDC is not configured. Set oidc.issuerURL and oidc.clientID to enable authentication.
